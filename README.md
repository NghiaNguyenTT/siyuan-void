# Void — A Minimal SiYuan Theme

A clean, borderless dark theme for [SiYuan Note](https://github.com/siyuan-note/siyuan).

## Design

- **Borderless** — no visible borders, panels separated by subtle shadows
- **Floating islands** — each panel floats above the canvas with soft elevation
- **DM Sans** — clean geometric sans-serif font, self-hosted
- **Dark canvas** — slightly lighter body background makes panel shadows visible
- **Minimal** — no distractions, pure focus on content

## Installation

### From SiYuan Marketplace
1. Open SiYuan → Settings → Appearance → Theme
2. Click the marketplace icon
3. Search for "Void"
4. Click Install

### Manual
1. Download `package.zip` from the latest [Release](https://github.com/NghiaNguyenTT/siyuan-void/releases)
2. Extract to `{workspace}/conf/appearance/themes/siyuan-void/`
3. Restart SiYuan or reload with `Ctrl+R`
